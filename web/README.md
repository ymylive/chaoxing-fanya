# 超星学习通 - Web前端

这是超星学习通自动化学习助手的Web前端界面，提供了更友好的用户交互体验。

## 功能特性

- 🔐 **用户登录** - 安全的账号密码登录
- 📚 **课程选择** - 可视化课程列表，支持单选或全选
- ⚙️ **灵活配置** - 播放倍速、并发数、未开放章节处理策略
- 📊 **实时进度** - 实时监控学习进度和任务状态
- 📝 **日志查看** - 查看详细的执行日志

## 技术栈

- **前端框架**: React 18
- **构建工具**: Vite
- **样式**: TailwindCSS
- **HTTP客户端**: Axios
- **图标**: Lucide React

## 快速开始

### 1. 安装依赖

```bash
cd web
npm install
```

### 2. 启动后端服务

在项目根目录下启动Flask后端：

```bash
cd ..
pip install -r requirements.txt
python app.py
```

后端将在 `http://localhost:5000` 运行

### 3. 启动前端开发服务器

```bash
cd web
npm run dev
```

前端将在 `http://localhost:3000` 运行，并自动代理API请求到后端。

### 4. 访问应用

在浏览器中打开 `http://localhost:3000`

## 构建生产版本

```bash
npm run build
```

构建的文件将输出到 `dist` 目录。

## 预览生产构建

```bash
npm run preview
```

## 使用说明

1. **登录** - 输入超星学习通的手机号和密码
2. **选择课程** - 在课程列表中选择要学习的课程（不选择将学习所有课程）
3. **配置参数** - 设置播放倍速、并发章节数等参数
4. **开始学习** - 点击"开始学习"按钮，系统将自动完成学习任务
5. **监控进度** - 实时查看学习进度和日志输出

## 配置项说明

- **播放倍速**: 视频播放速度，范围 1.0 - 2.0
- **并发章节数**: 同时处理的章节数量，建议 4-8
- **未开放章节处理**:
  - `retry`: 重试上一个任务点
  - `ask`: 询问用户是否继续
  - `continue`: 自动跳过

## 注意事项

- 确保后端服务正常运行
- 首次使用需要安装所有依赖
- 建议在稳定的网络环境下使用
- 请遵守学校规定，合理使用本工具
